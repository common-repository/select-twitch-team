== Description ==




Inserts your Twitch Stream Team roster (names, logos, banners, links) on any page on your wordpress site. As easy as dropping a shortcode on any page.



=== Installation ==

= 

Install the plugin 

= 

1. Download and unzip select-twitch-team archive contents to the `/wp-content/plugins/select-twitch-team` directory or add it using WordPress' plugin manager.

 2. Activate the plugin through the 'Plugins' menu in WordPress.

= Configure your team
 
= 

1. Go to *Twitch Team Select*

  2. Enter the name in *lowercase* of the team you want to use with the plugin.

  3. Click *Save Changes* use the *Show Team* button to populate Team Preview section.



= Add shortcode = 

= 

Go to any page and open text editor insert *[twitch_team_select]* that's it.



== Frequently Asked Questions ==

= 

I want to add a roster for my team but there is no way to set my team name. 

= 

You must first add your team in the plugin's settings page. Then, you will be able to use the shortcode *[twitch_team_select]* on any page.
